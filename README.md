Task Manager API:-Build a backend API for managing tasks and users. Each user can have multiple tasks. This will cover entities, relations, CRUD operations, and database integration.

Features:-
a.User Management:-
1.perform crud operation on User.
2.user fields:-user_id(primary key),name,email.

b.task Management:-
1.perform crud operation on task.
2.task fields:-task_id,description,status,user_id(foreign key).

Relationship:
one to many relationship.
one user has many task.
